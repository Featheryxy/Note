修改功能和扩展功能使用extends 

```java
class A{
	public void errorMethod(){
	
	}	
}

// 修改功能 
class B() extends A{
	@Override
	public void errorMethod(){
		// 覆盖原有功能
	}	
}

// 扩展功能
class B() extends A{
	@Override
	public void errorMethod(){
		super.errorMethod();
        // 新功能
	}	
}
```
